# Face-Detection
Python Real time Face Detection


### lib:

- opencv-Python
- deepface


#### opencv github + docs:

         https://github.com/opencv/opencv
         https://opencv.org/

#### deepface github + doc:

    https://github.com/serengil/deepface



## installation:

```bash
    pip install -r requirements.txt
```

## run:

```bash
    python app.py
```
